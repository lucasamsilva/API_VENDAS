# API Vendas Spring Boot

API desenvolvida utilizando Spring Boot.

Trata-se de uma API para vendas, com serviços de cadastro de usuário, cadastro de produtos com fotos utilizando Amazon S3 e envio automático de e-mails.

# Utilização

Faça o download do .zip do projeto ou realize um git clone. Importe o projeto para a IDE de sua preferência e altere os arquivos .properties localizados dentro de src/main/resources, preenchendo com seus dados os campos que estiverem em branco.


>É necessário uma conta de e-mail da google para o serviço de envio de e-mails.

>É necessário uma conta na AWS para a criação de um bucket Amazon S3, sendo necessário liberar o acesso público ao bucket.
